RailSmart

LIVE
HTML5 CSS3 Vanilla JS Indian Railways API

RailSmart is a cutting-edge train tracking and analytics dashboard designed for Indian Railways. Built with a modular JavaScript architecture and Leaflet.js, it offers real-time journey insights, live status updates, and interactive map-based tracking. The application facilitates seamless navigation through hash-based routing, allowing users to search for trains, monitor delays via a dedicated analytics engine, and visualize routes on a high-performance map. With a premium dark-themed interface, smooth CSS animations, and optimized typography, RailSmart transforms complex railway data into an intuitive, visually stunning experience, providing commuters with precise, easy-to-access information for smarter travel planning and journey management.

Key Working Principles:
• Modular Architecture: Uses a layered approach (Data → Utils → Pages → App) for scalable logic.
• Geospatial Tracking: Integrates Leaflet.js to render interactive maps with custom train markers and route waypoints.
• Dynamic State Management: Implements hash-based navigation to transition between Search, Live Status, and Analytics without page reloads.
• Data Visualization: Leverages a structured data layer to simulate real-time tracking and delay analytics.

Polluture

LIVE
HTML5 Vanilla JS Leaflet.js Chart.js CountUp.js

Polluture is an interactive, web-based ecological impact mapper designed to visualize the true environmental cost of urban development. Users draw custom zones on a real-world map and simulate land-use changes, like converting wetlands into built-up areas. By configuring land type, population density, building floors, and a time horizon, the tool uses deterministic formulas to instantly calculate localized impacts. It computes AQI deterioration, groundwater depletion, and temperature rise. Uniquely, Polluture quantifies these hidden environmental costs into a financial "Ecological Debt." Complete with dynamic charts, 3D building overviews, and narrative timelines, it powerfully illustrates the long-term ecological consequences of urbanization.

RideRiddles

LIVE
Node.js Express.js Leaflet.js OpenStreetMap Bidirectional A*

Ride Riddles is a sophisticated full-stack navigation platform tailored for Kolkata’s urban landscape. It leverages live OpenStreetMap data and a custom Bidirectional A* routing engine to calculate optimal paths. The app distinguishes itself through mode-aware logic, snapping routes to specific road types for cars, bikes, or pedestrians. Users can personalize journeys by prioritizing factors like road smoothness, shaded paths, or speed. By integrating real-world traffic levels and road conditions from a curated dataset, Ride Riddles provides highly accurate, context-aware travel estimates, ensuring a premium and efficient navigating experience for city commuters.

UEM SEATS

Premium Examination Seating System
LIVE
AI-Enhanced
HTML5 CSS3 Vanilla JS AI Algorithms

UEM SEATS is a state-of-the-art examination seating management system tailored for the University of Engineering & Management. It empowers faculty with a secure dashboard to upload routine and student databases via CSV. A sophisticated algorithm then dynamically allocates rooms and intelligently interleaves students from different streams, maximizing academic integrity. Faculty can also utilize manual overrides for fine-tuned seating control before finalizing allocations. For students, the seamless portal provides cutting-edge real-time identity verification powered by TensorFlow.js, issuing a digital exam ticket equipped with a unique QR code. Built entirely with vanilla technologies and LocalStorage, it boasts a premium, responsive glassmorphism aesthetic.

Movie-Heist

LIVE
React.js TMDB API CSS3 YouTube Embeds

Movie-Heist is a cinematic, AI-powered movie discovery and booking application built with Next.js and Tailwind CSS. Leveraging the TMDB API, it delivers real-time information on trending and currently playing movies through a premium, dark-themed glassmorphic interface enriched with fluid Framer Motion animations. The app simplifies the entertainment journey by offering smart search capabilities and genre-based filtering. For theatrical releases, it seamlessly provides accurate availability and direct "Book Now" links via BookMyShow. For home viewing, Movie-Heist intelligently connects users to over 20+ streaming platforms (like Netflix and Prime Video) while offering immersive in-app YouTube trailer playback.
