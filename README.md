# Arijit Pal - Personal Portfolio

A sleek, highly interactive personal portfolio website showcasing my technical skills, projects, and academic background. The design features a premium color palette, modern glassmorphism aesthetics, and a dynamic responsive sidebar.

## Features

* Futuristic Scroll-Interactive UI: Features a dedicated sidebar that tracks scroll progress and illuminates based on mouse cursor position.
* Masonry Gallery Layouts: Dynamic photo walls supporting multi-aspect natural image scaling.
* Advanced CSS Typography: Refined font stacks and elegant CSS typography utilizing Playfair Display and Inter font families.
* Responsive Design: Fully scales and collapses cleanly between desktop, tablet, and mobile views.
* Zero Dependencies: Built entirely with vanilla HTML, CSS, and modern JavaScript for maximum performance.

## Tech Stack

* HTML5 (Semantic Structure)
* CSS3 (Grid/Flexbox Layouts, Transitions, Keyframe Animations)
* Vanilla JavaScript (DOM Observers, Event Listeners)

## Local Development

To run the portfolio locally on your machine, launch a simple HTTP server in the repository root:

python -m http.server 8000

Then open your browser and navigate to http://localhost:8000
